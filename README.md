# Empório Liasch — Site (Next.js)

Premium marketing website for **Empório Liasch** (Americana/SP): an intimate wine & bistrô experience with reservations.

## Highlights
- Cinematic hero with image overlay and typography hierarchy
- Premium navbar (scroll state + mobile full-screen menu)
- Responsive layout and polished UI micro-interactions
- Built with modern, production-ready stack

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- Framer Motion
- Radix UI + lucide-react

```bash
npm install
npm run dev
